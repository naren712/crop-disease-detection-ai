# 🌿 Crop Disease Detection using Machine Learning and AI

This project is an AI-based crop disease detection system developed using Machine Learning, React JS, Django, and Gemini API.
Users can upload crop leaf images and the system predicts diseases with confidence and severity levels. The system also provides solutions, prevention methods, multilingual support, voice output, and AI-based question answering.

---

## 🚀 Features

- Crop disease prediction using CNN
- Confidence and severity analysis
- Solution and prevention suggestions
- AI-based question answering
- Tamil and English language support
- Voice output feature
- Temporary detection history
- Crop selection with details

---

## 🛠 Technologies Used

- React JS
- Django
- Python
- CNN (Convolutional Neural Network)
- Gemini API

---

## 📂 Dataset

- PlantVillage Dataset
- Around 280 crop images used for training

---

## 📊 Model Performance

- Training Accuracy: 95.83%
- Validation Accuracy: 83.73%
